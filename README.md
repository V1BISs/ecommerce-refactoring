# Задание для рефакторинга: Интернет-магазин

## Описание проекта

Этот проект представляет собой простой интернет-магазин, написанный на React с использованием Redux Toolkit. Код содержит типичные антипаттерны, которые я исправил.

## Цель задания. Ваша задача - провести рефакторинг кода, улучшив его качество, читаемость и поддерживаемость. 

# Отчет о рефакторинге проекта "Интернет-магазин"

## Основные проблемы, которые были решены:

1. Монолитная структура компонентов — весь код находился в одном большом файле
2. Отсутствие модульности и переиспользуемости
3. Потенциальные проблемы с производительностью — отсутствие оптимизации ререндеров
4. Сложность поддержки и масштабирования — кодовая база трудна для понимания и расширения
5. "Вермишель" импортов — неорганизованные пути импортов

### Разделение монолитного кода на компоненты
- Выделены переиспользуемые UI-компоненты (ProductCard, CartItem, Button)
- Созданы контейнеры для бизнес-логики (ProductListContainer, CartContainer)
- Разделена логика представления и состояния

### Оптимизация Redux логики
- Вынесение экшенов в отдельные модули для переиспользования
- Централизованное управление типами экшенов (предотвращение опечаток)
- Упрощение создания и dispatch экшенов

### Оптимизация производительности React
- Обработчики событий обёрнуты в useCallback для предотвращения лишних ререндеров
- Оптимизация списков с помощью ключей

### Улучшение системы импортов
- Настройка vite.config.js для использования абсолютных импортов
- Создание файлов index.js в каждой папке для barrel exports

### Вынос данных в отдельные модули
- Моковые данные товаров вынесены в src/mocks/products.js
- Конфигурационные данные (фильтры, настройки) вынесены в отдельные модули

### Вывод
Проведённый рефакторинг преобразовал монолитное приложение в современную, масштабируемую архитектуру 
с чётким разделением ответственности. 
Код стал чище, понятнее и легче в поддержке, что особенно важно для командной разработки и дальнейшего развития проекта.

## Функциональность

Приложение включает:
- Каталог товаров с фильтрацией и поиском
- Корзину покупок
- Добавление/удаление товаров
- Изменение количества товаров
- Оформление заказа

## Инструкции по запуску

1. Установите зависимости:
npm install

2. Запустите проект:
npm run dev

3. Откройте браузер по адресу http://localhost:5173
